pick(dc, tileList, pickDelegate)

Causes this terrain to perform the picking operations on the specified tiles, as appropriate for the draw context's pick settings. Normally, this draws the terrain in a unique pick color and computes the picked terrain position. When the draw context is set to region picking mode, this omits the computation of a picked terrain position.
Parameters:
Name Type Description
dc DrawContext The current draw context.
tileList Array The list of tiles to pick.
pickDelegate Object Indicates the object to use as the picked object's userObject. If null, then this tessellator is used as the userObject.
