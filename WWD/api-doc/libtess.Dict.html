Class: Dict

Dict

new Dict(framenon-null, leq)

A list of edges crossing the sweep line, sorted from top to bottom. Implementation is a doubly-linked list, sorted by the injected edgeLeq comparator function. Here it is a simple ordering, but see libtess.sweep for the list of invariants on the edge dictionary this ordering creates.
