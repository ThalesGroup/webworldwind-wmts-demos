Class: ElevationImage

ElevationImage

Holds elevation values for an elevation tile. This class is typically not used directly by applications.

pixel(x, y) → {Number}

Returns the pixel value at a specified coordinate in this elevation image. The coordinate origin is the image's lower left corner, so (0, 0) indicates the lower left pixel and (imageWidth-1, imageHeight-1) indicates the upper right pixel. This returns 0 if the coordinate indicates a pixel outside of this elevation image.
