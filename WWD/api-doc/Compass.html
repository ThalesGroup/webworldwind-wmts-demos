Class: Compass

Compass

Displays a compass image at a specified location in the World Window. The compass image rotates and tilts to reflect the current navigator's heading and tilt.

Constructor

new Compass(screenOffset, imagePath)

Constructs a compass.
Parameters:
Name Type Description
screenOffset Offset The offset indicating the image's placement on the screen. If null or undefined the compass is placed at the upper-right corner of the World Window. Use the image offset property to position the image relative to the screen point.
imagePath String The URL of the image to display. If null or undefined, a default compass image is used.
