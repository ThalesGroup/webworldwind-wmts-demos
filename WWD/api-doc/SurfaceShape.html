Class: SurfaceShape

SurfaceShape

Represents a surface shape. This is an abstract base class and is meant to be instantiated only by subclasses.

Surface shapes other than SurfacePolyline SurfacePolyline have an interior and an outline and utilize the corresponding attributes in their associated ShapeAttributes ShapeAttributes. They do not utilize image-related attributes.
