Class: WmsLayerCapabilities

WmsLayerCapabilities

Represents a WMS layer description from a WMS Capabilities document. This object holds all the fields specified in the associated WMS Capabilities document.

Constructor

new WmsLayerCapabilities(layerElement, parentNode)

Constructs an WMS Layer instance from an XML DOM.
Parameters:
Name Type Description
layerElement Object A WMS Layer element describing the layer.
parentNode Object An object indicating the new layer object's parent object.
